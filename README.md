## 1. Bayesian Optimization-Driven Random Forest Model for Crop Recommendation
**Journal:** SN Computer Science (SNCS), Springer  
**Status:** Under Peer Review  
**Domain:** Machine Learning, Agricultural AI  

**Abstract:**  
This paper presents a crop recommendation system leveraging Random Forest 
classification optimized via Bayesian Hyperparameter Tuning. The model was 
trained and evaluated on structured agricultural datasets incorporating climate 
and soil parameters, achieving 99.78% accuracy. The study demonstrates the 
effectiveness of automated hyperparameter optimization in improving ML model 
performance for precision agriculture applications.

## 2. Emotion-Driven Student Engagement Detection System
**Journal:** The Visual Computer, Springer  
**Status:** Under 2nd Revision  
**Domain:** Computer Vision, Deep Learning, EdTech AI  

**Abstract:**  
This paper proposes a vision-based AI system for real-time student engagement 
detection using facial emotion recognition. The system employs CNN-based facial 
feature extraction combined with ML classifiers to map emotional states to 
engagement metrics, capable of handling multiple faces simultaneously in 
live classroom environments.

📄 [View Paper](./Student_Engagement_Detection_VisualComputer_2025.pdf)
